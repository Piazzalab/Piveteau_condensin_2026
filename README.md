# Piveteau_condensin_2026
Repository of scripts and reference genomes used in Piveteau et al. EMBO J. 2026

Conda environment .yml files are provided to install the four environments required to run the scripts. 

Compressed files contain the reference fasta genomes and their bowtie2 indexes.

All the .ipynb Jupyter notebook run in the jupyter_analysis conda environment.

The Script_reads_to_hic.sh, DistanceLaw1_remove_mito_and_2_micron.sh and Plot_matrix_using_hicstuff.sh runs in the hicstuff conda environment.

The Chromosight scripts run in the chromosight conda environment.
